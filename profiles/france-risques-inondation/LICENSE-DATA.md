## Informations complémentaires

- **Hub'Eau** : API publique du Système d'Information sur l'Eau (SIE). Données de référentiel des stations hydrométriques, actualisées quotidiennement.
- **Géorisques** : Portail de prévention des risques naturels du Ministère de la Transition Écologique. Les couches TRI (Territoires à Risques Importants d'inondation) sont issues de la Directive Inondation européenne 2007/60/CE.
- **BD TOPO®** : Base de données topographique de l'IGN. Couche cours d'eau extraite via l'OGC API Features.
